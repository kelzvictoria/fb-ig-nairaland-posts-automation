## About The Project

This is a tool for fetching content on Yo! youtube channel and programmatically posting to gowithyo.com(wordpress site), nairaland, facebook, and instagram.

### Built With

Node JS

### Code Entry Point

app.js

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com.pmglobaltechnology.com/kelzvictoria/yo-new-post-script
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
